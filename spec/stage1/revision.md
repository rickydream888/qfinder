# BUG修改记录

## BUG-1

```
[ERROR]
Tauri core API is not available
```

Tauri 2 默认不再向页面注入 window.__TAURI__，需要显式开启 withGlobalTauri。

## BUG-2

用户的家目录下显示了Application Data、Local Settings、My Documents、NetHood等无法打开的目录或文件。

## BUG-3

拖动目录或文件的功能完全不可用：

- 拖动文件时无动画效果，被拖动的目录或文件并未创建任何元素随鼠标移动；
- 拖动时鼠标不管放到什么地方（包括可以合法移动的目录）上都会变成不能操作的样式。

## BUG-4

Windows平台上展开过驱动器C:之后，就无法展开用户的家目录了。

## BUG-5

在重命名时按下Delete键会触发删除文件的操作。

## BUG-6

Windows下对网络驱动器中的文件进行预览时会非常卡。

通过异步操作，加入载入动画等方法优化。

# Feature调整

## FTR-1

用户的家目录直接显示为“家目录”，应该显示为用户的账号。

## FTR-2

预览模式时应该可以通过拖动中间那个分割线的方式调整左边文件系统树和右边预览区的占比。文件系统树部分宽度太小时应出现横向滚动条。

## FTR-3

复制后粘贴、剪切后粘贴或移动时目标位置有同名的目录或文件时：

- 同名的目录：弹框提示用户选择“合并目录”、“替换目录”和“取消”，合并目录等于把源目录中的文件都复制或移动到目标处的同名目录里，有同名的文件则覆盖，替换目录等于删除目标处的同名目录，把源目录复制或移动到目标处；
- 同名的文件：弹框提示用户选择“替换文件”和“取消”。
